# School Result App 

This is a simple school result calculator created using HTML, CSS, and JavaScript.

## Features
-Add student name and subject marks
-Automatic calculation of: Total Marks, Average Marks, Grade (A / B / C / Fail)
-Dynamic table row creation
-Delete button for each row
-Input fields clear after submission
-Dark and light mode
-Calculate total student and Pass
-Responsive design
-Clean UI with styled table

## Technologies Used
- HTML
- CSS
- JavaScript
